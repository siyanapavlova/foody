The project is still under development

=========================================================

To run the application:

You need to have Python 2.7 installed

The following libraries may need to be installed using pip install too:
watson_developer cloud
flask
flask_wtf

=========================================================

To run the application with a text user interface:

1) Open a terminal
2) Navigate to the application folder
3) Open the app folder
4) Type 'python test.py'

=========================================================

To run the graphic user interface:

1) Open the terminal
2) Navigate to the application folder
3) Type 'python run.py'
4) Open a browser and go to http://127.0.0.1:5000/message
